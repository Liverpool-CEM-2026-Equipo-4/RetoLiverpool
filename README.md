# Descripción del proyecto
Se diseñó una aplicación web basada en inteligencia artificial para anticipar riesgos relacionados con la renovación de marcas. A través del análisis de información histórica, legal y comercial, la plataforma permite identificar patrones, predecir la probabilidad de renovación de una marca y brindar apoyo a la toma de decisiones estratégicas. Además, integra capacidades de consulta inteligente mediante agentes especializados y búsqueda semántica sobre documentación relevante, facilitando el acceso rápido a información clave sobre litigios.
